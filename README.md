# HUMAN COMPUTER INTERACTION - HCI - LABORARY 1
Using HCI Guidelines to create a log in page.

### Project Uses the following
* ReactJs + TypeScript
* Tailwind CSS
* Shadcn Components

### For Installation
```
npm install
cd LOGIN_FORM_HCI
npm run dev
```



[WPESFMS Login Page Demo](https://wpesfmslogindemo.vercel.app)

**Log in credentials than can be used to log in can be found in public/users.json**

Example : 
email: example@example.com,
password: examplepassword
